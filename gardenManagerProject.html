Garden Manager

GitHub Code
Live Link

The Community Garden Management App is a innovative and socially impactful platform designed to foster a sense of community and promote local sourcing of food. This app aims to bring neighborhoods closer together through the joy of gardening.

Tech Used

  • Java with Springboot Framework: Used for the backend development, providing a robust and streamlined development experience.
  • MySQL with Spring Security: Employed for data storage and secure authentication and authorization of users.
  • Ajax: Enabled asynchronous communication with the backend for seamless data retrieval.
  • Full Calendar Plugin: Utilized for visually displaying and managing events on a calendar for garden managers.
  • JSPs with Spring MVC: Rendered front-end view pages, maintaining a clear separation of data and application logic.
  • CSS, Bootstrap, and Tailwind: Styled the user interface for an attractive and responsive design.
  • JavaScript: Implemented interactivity and dynamic behavior on the client-side.
  • Agile Development Process: Followed to ensure iterative and customer-centric development.
